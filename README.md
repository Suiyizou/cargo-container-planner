# 货代装箱计算器 B/S 前端版

这是重新实现的无数据库 B/S 原型。当前版本只有前端页面和一个本地静态服务，数据运行时保存在 `Map` 中，并同步到浏览器 `localStorage`。

## 运行

```bat
cd C:\Users\ASUS\Documents\Playground\cargo-container-planner
npm install
node dev-server.js
```

浏览器访问：

```text
http://127.0.0.1:5177
```

## 当前能力

- 录入货物长、宽、高、数量、单重和摆放规则。
- 按物理尺寸将货物展开为单件箱体，并尝试放入集装箱。
- 箱型排序卡片可直接点击切换当前箱型，按预计箱数和利用率排序。
- 箱型卡片采用两行排列，避免横向过长；计算和视图切换时提供加载提示。
- Three.js 主视图中用不同颜色标注不同货物，可拖动旋转到俯视角；鼠标滚轮保留给页面滚动。
- 主视图内置货物图例、箱型长宽高尺寸卡片和三维尺寸标注线，鼠标悬停货物可查看明细。
- 顶视图/正视图/侧视图使用 Three.js 正交相机，可拖动平移。
- 顶部工具栏提供货物录入弹窗、数据操作菜单和算法说明页面。
- 当一个箱型需要多个货舱时，左侧结果栏会显示第 1 箱、第 2 箱等切换按钮，主视图展示当前选中的货舱。
- 支持添加自定义箱型，并参与推荐计算。
- 支持普通旋转、保持朝上、不可重压、托盘/异形等基本规则。
- 比较多个箱型，给出推荐箱型和预计箱数。
- 无数据库：`Map<string, Cargo>` + `localStorage`。

## 说明

当前装箱算法是前端启发式 3D bin packing：

- 货物不允许重叠。
- 货物不能超过箱体长、宽、高。
- 普通货物允许旋转，保持朝上/托盘货不旋转。
- 不可重压货物放置后不会在其上方继续生成堆叠空间。
- 如果一个箱放不完，会继续估算第二个、第三个箱。

它适合做业务计算原型和箱型选择辅助，不等同于专业排柜软件的最优三维装箱求解器。

## 立体图方案

当前使用本地安装的 Three.js，不依赖外网 CDN。页面通过 import map 加载 `node_modules/three`，仍然是无数据库的本地 B/S 原型。
